= AI Computer Vision at the Edge

++++
<span class="workshop-time-badge">~20 min</span>
++++

image::13-ai-computer-vision.png[AI Computer Vision at the Edge — NeuroFace PPE detection,960]

**AI Computer Vision at the Edge** is the primary demo pattern of this workshop. **NeuroFace** is the reference application — combining YOLO PPE detection, face recognition, object detection, MaaS chat, and federated edge inference across a hybrid mesh.

== Pattern architecture

.AI CV at the Edge — middleware footprint
[cols="1,2", options="header"]
|===
| Product | NeuroFace integration

| OSSM3 / Istio | Ambient mesh (hub neuroface ns), sidecar (CV gateway)
| Skupper | CV federation + Kafka tunnel (east cv-kafka)
| Kafka / Strimzi | PPE events `cv.ppe.detections` (producer + Camel K consumer)
| OpenShift AI / KServe | YOLO PPE InferenceService + workbenches
| Kuadrant / AI Gateway | Documented MaaS alternative (module 23)
| Camel K | PPE Kafka→Mailpit violation alerts
| OpenTelemetry / Tempo | App-level FastAPI + mesh traces
| Grafana | Hub NeuroFace + CV journey dashboards
| ACM / GitOps | Argo CD managed apps
| Vault + ESO | MaaS API key sync
| DevSpaces | Hub DevSpaces — open NeuroFace code from Developer Hub
| Quay | Image registry (`neuroface-backend`, `neuroface-frontend`, `neuroface-ppe-serving`)
| GitLab | Scaffolder target (AI CV software template)
| Developer Hub | Full system catalog + `ai-computer-vision` software template
| Kubecost | Namespace cost labels (`kubecost.io/team: neuroface`)
| Mailpit | PPE alert emails
|===
