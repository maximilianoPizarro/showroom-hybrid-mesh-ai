== Sync hero images from platform

From this repo (sibling checkout):

[source,bash]
----
bash ../hybrid-mesh-platform/scripts/sync-showroom-content.sh
----

From `hybrid-mesh-platform`:

[source,bash]
----
SHOWROOM_DIR=../showroom-hybrid-mesh-ai bash scripts/sync-showroom-content.sh
git -C ../showroom-hybrid-mesh-ai add content/modules/en/modules/ROOT/images/
----

Hero PNGs are **live hub cluster captures** — see `hybrid-mesh-platform/scripts/workshop-screenshot-manifest.yaml`. Preserve ACS heroes `03` and `20` (do not overwrite).

== Publish to cluster

[source,bash]
----
git push origin main
oc rollout restart deployment/showroom -n showroom
oc rollout status deployment/showroom -n showroom --timeout=120s
----

If the showroom route returns 503, run `bash scripts/apply-workshop-showroom.sh` from the platform repo.
