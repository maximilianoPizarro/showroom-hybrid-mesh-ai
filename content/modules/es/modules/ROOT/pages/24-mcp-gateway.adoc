== Qué harás

Verifica MCP Gateway en `mcp-system`, usa **Developer Hub → Lightspeed** con tools MCP, y registra MCP server OpenShift AI en el dashboard.

== Panorama


NOTE: **Ritmo del taller (~4 h hands-on)** — Cada módulo incluye pasos **Solo visualización** (~5–10 min) y **Hands-on** (lab completo). Omite Hands-on para un tour ejecutivo ~90 min.

**MCP Gateway** despliega CRDs Kuadrant, controlador, Gateway API, **ArgoCD MCP** y **k8s MCP** — patrón test-drive-pe-oscg. URL: `https://mcp-gateway.%HUB_DOMAIN%/mcp`.

**Solo visualización (~8 min):** Catálogo → **workshop-mcp-gateway**; demo Lightspeed.

**Hands-on (~30 min):** **/lightspeed** — actividad: listar apps Argo CD OutOfSync y proponer sync.
