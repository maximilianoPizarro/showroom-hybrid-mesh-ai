== What you will do

Open link:https://neuroface.%HUB_DOMAIN%[NeuroFace] and **Developer Hub → Catalog → neuroface-workshop**. Enable webcam for **PPE Safety** detection; chat uses MaaS at `/api/chat`. == Overview

NOTE: **Workshop pacing (~4 hours hands-on)** — Each module includes **Overview-only** steps (demo/visualization, ~5–10 min) and **Hands-on** steps (full lab). Skip Hands-on blocks to run a ~90-minute executive tour.

NeuroFace combines **YOLO PPE serving** (`yolo-ppe-serving`, classes hardhat / safety-vest / goggles), optional **OVMS** webcam face detection, and **MaaS chat** at `/api/chat`. The MaaS API key is stored in secret **`neuroface-maas-api-key`** (never committed to Git).

**Overview-only (~10 min):** Developer Hub → **neuroface-workshop** → Topology; open UI without webcam.

**Hands-on (~30 min):** Test webcam PPE Safety + chat; run the **PPE Workbench** notebook; verify secret and deployments in namespace `neuroface`.

== Hands-on: PPE Workbench (image prediction)

Use the pre-loaded Jupyter notebook to detect PPE on uploaded factory-floor images.

. **Start the workbench** — OpenShift Console → **AI** → **Workbenches** → namespace `neuroface` → **ppe-workbench** → **Start**.
. **Open the notebook** — Either:
+
--
* OpenShift AI dashboard → open **ppe-detection.ipynb**, or
* Direct route: link:https://ppe-workbench.%HUB_DOMAIN%[ppe-workbench.%HUB_DOMAIN%] (after workbench is running)
--
+
. **Run cells in order:**
+
--
. Setup — verifies `PPE_ENDPOINT` health and NeuroFace `/api/ppe/status`
. Upload — place a `.jpg` in `uploads/sample.jpg` (or change `IMAGE_PATH`)
. Detect — calls `POST /v1/predict` on `yolo-ppe-serving`
. Visualize — draws bounding boxes for hardhat, safety-vest, goggles
. Compare (optional) — local Ultralytics model vs serving endpoint
--
+
. **Expected output** — JSON detections with `class_name`, `confidence`, and `bbox` fields; matplotlib overlay on the image.

[source,bash]
----
# Facilitator / terminal checks
curl -sk https://neuroface.%HUB_DOMAIN%/api/ppe/status
curl -sk https://ppe-workbench.%HUB_DOMAIN%/  # 200 when workbench pod is running
oc get deploy yolo-ppe-serving -n neuroface
oc get notebook ppe-workbench -n neuroface
----
