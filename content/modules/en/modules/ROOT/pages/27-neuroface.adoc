== What you will do

Open link:https://neuroface.%HUB_DOMAIN%[NeuroFace] and **Developer Hub → Catalog → neuroface-workshop**. Enable webcam for **PPE Safety** detection; chat uses MaaS at `/api/chat`. Facilitator must inject MaaS key via `scripts/apply-maas-secrets.sh` (secret **`neuroface-maas-api-key`**).

== Overview

NOTE: **Workshop pacing (~4 hours hands-on)** — Each module includes **Overview-only** steps (demo/visualization, ~5–10 min) and **Hands-on** steps (full lab). Skip Hands-on blocks to run a ~90-minute executive tour.

NeuroFace combines **OVMS local vision** (`ovms.enabled: true`, model **`face-detection-retail-0005`** for retail/PPE face detection) and **MaaS chat** at `/api/chat`. The MaaS API key is stored in secret **`neuroface-maas-api-key`** (never committed to Git).

**Overview-only (~10 min):** Developer Hub → **neuroface-workshop** → Topology; open UI without webcam.

**Hands-on (~30 min):** Test webcam PPE Safety + chat; verify secret and deployment in namespace `neuroface`.

*Helm values* — NeuroFace chart on hub:

[source,yaml]
----
# charts/all/neuroface/values.yaml
ovms:
  enabled: true
  modelName: face-detection-retail-0005   # PPE Safety / retail face detection
chat:
  enabled: true
  modelEndpoint: https://maas-rhdp.apps.maas.redhatworkshops.io/v1
  modelName: llama-scout-17b
route:
  enabled: true
  host: neuroface.%HUB_DOMAIN%
----

*MaaS secret* — injected post-install:

[source,yaml]
----
# charts/all/neuroface/templates/secret-neuroface-maas-api-key.yaml
apiVersion: v1
kind: Secret
metadata:
  name: neuroface-maas-api-key
  namespace: neuroface
stringData:
  api-key: <from RHDP litemaas or apply-maas-secrets.sh>
----

Facilitator:

[source,bash]
----
export MAAS_KEY_LLAMA='sk-...'
bash scripts/apply-maas-secrets.sh
oc rollout restart deployment/neuroface -n neuroface
----

== Features, benefits & cloud configuration

=== Key features

* **OVMS** **`face-detection-retail-0005`** for low-latency webcam PPE Safety demo.
* **MaaS chat** at `/api/chat` for operator Q&A.
* Catalog entity **neuroface-workshop** links UI, routes, and OpenShift AI workbench **`neuroface-ml-lab`**.

=== Business benefits

* Vision at the app edge; generative answers from governed hub MaaS.
